# AI Resume Builder

## Project Overview
AI Resume Builder is a web application that helps users create professional resumes using customizable templates and real-time preview. Users can enter their details, select templates, and download resumes in PDF format.

This project was developed using modern front-end technologies with a focus on responsive design and usability.

---

## Features
- Create and edit resume content dynamically
- Multiple resume templates
- Real-time preview while editing
- Download resume as PDF
- Responsive UI for desktop and mobile devices

---

## Technologies Used
- React
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
- HTML & CSS
- JavaScript
